================================================================================
                    ЯК ПІДНЯТИ РІВЕНЬ ГІЛЬДІЇ
================================================================================

ЩО ТАКЕ РІВЕНЬ ГІЛЬДІЇ?
-----------------------
Рівень гільдії показує її розвиток та вплив.
Чим вищий рівень - тим більше можливостей має гільдія.

================================================================================
1. ЯК ПРАЦЮЄ СИСТЕМА РІВНІВ
================================================================================

ФОРМУЛА:
--------
• 1 рівень = 0 XP (початковий)
• 2 рівень = 1000 XP
• 3 рівень = 2000 XP
• 4 рівень = 3000 XP
• 5 рівень = 4000 XP
• N рівень = (N-1) × 1000 XP

ПРИКЛАД:
--------
Якщо у гільдії 2500 XP:
• Рівень = 1 + (2500 / 1000) = 3 рівень
• Залишилось до 4 рівня: 500 XP

================================================================================
2. ЯК ОТРИМАТИ XP ГІЛЬДІЇ
================================================================================

СПОСІБ 1: Внесок монетами (/contribute)
----------------------------------------
Команда: /contribute <сума>
• 1 монета = 1 XP
• Приклад: /contribute 1000 → +1000 XP

СПОСІБ 2: Захоплення територій
-------------------------------
Команда: /guild_capture <назва>
• Захоплення території: +500 XP
• Кожна територія приносить XP щодня

СПОСІБ 3: Перемога у війнах
----------------------------
Команда: /guild_war_battle
• Перемога в битві: +200 XP
• Захоплення території: +500 XP

СПОСІБ 4: Перемога над босом
-----------------------------
Команда: /guild_boss_attack
• Участь в бою: +50 XP
• Перемога над босом: +1000 XP

СПОСІБ 5: Щоденна активність
-----------------------------
• Кожен учасник в чаті: +1 XP/день
• Активні учасники: +5 XP/день

================================================================================
3. ПЕРЕВАГИ РІВНЯ ГІЛЬДІЇ
================================================================================

РІВЕНЬ 1-2 (Початковий):
------------------------
• Можливість створити гільдію
• До 5 учасників
• 0-1000 XP

РІВЕНЬ 3 (Середній) - МІНІМУМ ДЛЯ ВІЙН:
----------------------------------------
• Можливість захоплювати території
• До 10 учасників
• 2000+ XP
• Доступ до гільдійних воєн

РІВЕНЬ 4-5 (Просунутий):
------------------------
• До 20 учасників
• 3000-5000 XP
• Більше бонусів
• Сильніші воїни

РІВЕНЬ 6-10 (Елітний):
----------------------
• До 50 учасників
• 5000-10000 XP
• Унікальні бонуси
• Престиж в чаті

РІВЕНЬ 10+ (Легендарний):
-------------------------
• Максимальна кількість учасників
• 10000+ XP
• Найкращі бонуси
• Топ в рейтингу

================================================================================
4. ШЛЯХИ ПІДВИЩЕННЯ РІВНЯ
================================================================================

ШВИДКИЙ ШЛЯХ (1-3 дні):
-----------------------
1. Зберіть 1000-2000 монет
2. /contribute 2000 - внесіть монети
3. +2000 XP = +2 рівні
4. Витрати: 2000 монет

СЕРЕДНІЙ ШЛЯХ (1 тиждень):
--------------------------
1. Захопіть 2-3 території
2. Кожна дає +500 XP = 1000-1500 XP
3. Активність учасників: +50 XP/день
4. Разом: +1500-2000 XP за тиждень

ПОВІЛЬНИЙ ШЛЯХ (2-4 тижні):
---------------------------
1. Щоденна активність в чаті
2. Участь в битвах з босами
3. Перемоги в війнах
4. Поступове накопичення XP

================================================================================
5. ПРИКЛАДИ ПІДВИЩЕННЯ
================================================================================

ПРИКЛАД 1: Швидке підвищення
-----------------------------
Початковий рівень: 1
Мета: Рівень 3 (для воєн)

Дії:
1. /contribute 2000
2. Отримано: +2000 XP
3. Новий рівень: 3
4. Час: 1 хвилина
5. Витрати: 2000 монет

ПРИКЛАД 2: Захоплення територій
--------------------------------
Початковий рівень: 2
Мета: Рівень 5

Дії:
1. Захопити 3 території: +1500 XP
2. Перемога над босом: +1000 XP
3. Активність (7 днів): +350 XP
4. Разом: +2850 XP
5. Новий рівень: 5
6. Час: 1 тиждень

ПРИКЛАД 3: Командна робота
---------------------------
Початковий рівень: 3
Мета: Рівень 10

Дії:
1. 10 учасників × 50 XP/день = 500 XP/день
2. Захоплення 6 територій: +3000 XP
3. Перемоги в війнах (10): +2000 XP
4. Боси (5 перемог): +5000 XP
5. Разом: +13500 XP за 2 тижні
6. Новий рівень: 16

================================================================================
6. ПОРАДИ ДЛЯ ШВИДКОГО ПІДВИЩЕННЯ
================================================================================

1. ЗАЛИШАЙТЕСЯ АКТИВНИМИ:
   • Щоденна активність в чаті
   • Участь в івентах
   • Допомога іншим учасникам

2. ЗАХОПЛЮЙТЕ ТЕРИТОРІЇ:
   • Кожна територія = +500 XP
   • Дохід з територій = монети для внеску
   • 6 територій = 3000 XP

3. БИЙТЕ БОСІВ:
   • Кожен бос = +1000 XP
   • Спавн кожні 24 години
   • Командна робота

4. ВІЙНИ:
   • Перемоги = XP + території
   • Активна участь = більше XP
   • Співпраця з іншими гільдіями

5. ВНЕСОК МОНЕТАМИ:
   • 1 монета = 1 XP
   • Найшвидший спосіб
   • Використовуйте при потребі
